## Production Workflow

1. **Plan episode** → Use `templates/episode_outline.md`
2. **Draft scenes** → Use `templates/scene_card.md`
3. **Check continuity** → Reference `03_continuity_bible.md`
4. **Write heat** → Use `05_heat_beat_library.md` modules
5. **Polish voice** → Check against `04_voice_style_guide.md`
6. **Publish prep** → Use `06_metadata_pack.md` for blurbs/keywords
